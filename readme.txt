Shortcode for the ScanCircle scan button on partner websites. Only for registered ScanCircle partners.

== Description ==

Insert the following shortcode in your posts:

       [scancircle partner="partner-code" language="en/nl" scanmode="D|A|M|P" info="1" prompt="text" require="text" parameter="url-parameter" reference="fixed-value"]

The only required field is partner. Default language is "en" (English). Login to you ScanCircle partner page and click the wizard wand to see all available options.

== Installation ==

Manual installation:
1. Upload the folder `scancircle` to the `/wp-content/plugins/` directory
1. Activate the `scancircle` plugin through the 'Plugins' menu in WordPress
1. Place the [scancircle ...] shortcut in your posts
